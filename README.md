How to setup ?

1. install the 'model' folder here https://binusianorg-my.sharepoint.com/personal/rio002_binus_ac_id/_layouts/15/guestaccess.aspx?share=Evm3AvBImkJGiL_foylCBogBL9lcfownLgnWkejPcR5A9A&e=xC5dEt
2. put the model folder inside the app folder (beside the main.py)
3. open main.py, change every path directory ('model' folder, 'config-presets' folder, and 'pdf_files' folder)
4. on the terminal type ```pip install tkinter pdfplumber requests llama-cpp```
5. and type ```python main.py``` to start the app
